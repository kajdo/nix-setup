# Home-Manager Dotfiles Migration Plan

**Goal:** Migrate all stowed dotfile configurations to home-manager, eliminating the need for a separate `stow` step when setting up a new machine.

**Source:** `~/git/dotfiles/` (stow packages) — READ ONLY, for reference
**Target:** `~/git/nix-setup/nixos/home-manager/config/` — Files must be COPIED here

> ⚠️ **IMPORTANT:** All config files must be copied into the nix-setup repo to make it standalone. The repo should work on a fresh machine clone without requiring access to the dotfiles repo.

**Current config directory contents:**
```

├── starship/   ✅ already migrated
```

## Phase 2: GTK Theme Configuration

### [ ] 2.1 GTK2/GTK3 unified theming
**Status:** Currently stowed via `gtk-config` package, also `~/.gtkrc-2.0` exists in home directory

**Current files:**
- `~/.gtkrc-2.0` — GTK2 settings (managed by LXAppearance, has "DO NOT EDIT" warning)
- `~/git/dotfiles/gtk-config/.config/gtk-3.0/settings.ini` — GTK3 settings (stowed)

**Step 1 — Copy config files (for reference/backup):**
```bash
mkdir -p /home/kajdo/git/nix-setup/nixos/home-manager/config/gtk
cp ~/git/dotfiles/gtk-config/.gtkrc-2.0 /home/kajdo/git/nix-setup/nixos/home-manager/config/gtk/gtkrc-2.0
cp ~/git/dotfiles/gtk-config/.config/gtk-3.0/settings.ini /home/kajdo/git/nix-setup/nixos/home-manager/config/gtk/settings.ini
```

**Note:** These are for reference — the actual config will be generated from Nix attrsets

**Step 2 — Create `home-manager/modules/gtk-theming.nix`:**
```nix
{ config, pkgs, ... }:

{
  gtk = {
    enable = true;
    
    # GTK2 configuration (generates ~/.gtkrc-2.0)
    gtk2 = {
      enable = true;
      force = true;  # Overwrite existing ~/.gtkrc-2.0 (important!)
      
      theme = {
        name = "Adwaita-dark";
        package = pkgs.gnome-themes-extra;
      };
      
      iconTheme = {
        name = "Papirus-Dark";
        package = pkgs.papirus-icon-theme;
      };
      
      font = {
        name = "Sans Bold 10";
      };
      
      extraConfig = {
        gtk-cursor-theme-size = 0;
        gtk-toolbar-style = "GTK_TOOLBAR_BOTH_HORIZ";
        gtk-toolbar-icon-size = "GTK_ICON_SIZE_LARGE_TOOLBAR";
        gtk-button-images = 0;
        gtk-menu-images = 0;
        gtk-enable-event-sounds = 1;
        gtk-enable-input-feedback-sounds = 1;
        gtk-xft-antialias = 1;
        gtk-xft-hinting = 1;
        gtk-xft-hintstyle = "hintfull";
        gtk-xft-rgba = "none";
      };
    };
    
    # GTK3 configuration (generates ~/.config/gtk-3.0/settings.ini)
    gtk3 = {
      extraConfig = {
        gtk-theme-name = "Adwaita-dark";
        gtk-icon-theme-name = "Papirus-Dark";
        gtk-font-name = "Sans 10";
        gtk-cursor-theme-size = 0;
        gtk-toolbar-style = "GTK_TOOLBAR_BOTH_HORIZ";
        gtk-toolbar-icon-size = "GTK_ICON_SIZE_LARGE_TOOLBAR";
        gtk-button-images = 0;
        gtk-menu-images = 0;
        gtk-enable-event-sounds = 1;
        gtk-enable-input-feedback-sounds = 1;
        gtk-xft-antialias = 1;
        gtk-xft-hinting = 1;
        gtk-xft-hintstyle = "hintmedium";
        gtk-xft-rgba = "none";
      };
    };
    
    # Shared theme settings (fallback for gtk2/gtk3 if not explicitly set)
    iconTheme = {
      name = "Papirus-Dark";
      package = pkgs.papirus-icon-theme;
    };
    
    theme = {
      name = "Adwaita-dark";
      package = pkgs.gnome-themes-extra;
    };
  };
}
```

**Step 3 — Import in `home.nix`:**
Add `./modules/gtk-theming.nix` to imports

**Verification:** 
- `~/.gtkrc-2.0` exists and contains correct settings
- `~/.config/gtk-3.0/settings.ini` exists and contains correct settings
- GTK apps use correct theme
- `gsettings get org.gnome.desktop.interface gtk-theme` returns `Adwaita-dark`

**Important note about `gtk2.force = true`:**
- This overwrites any existing `~/.gtkrc-2.0` file
- Your current file has a comment "DO NOT EDIT! This file will be overwritten by LXAppearance"
- With home-manager, the file is now managed declaratively — LXAppearance edits will be lost on rebuild

**Cleanup:**
```bash
# Remove stowed package
stow -D -d ~/git/dotfiles gtk-config

# Remove generated config files (will be regenerated by home-manager)
# Note: ~/.gtkrc-2.0 will be overwritten by home-manager (force = true)
rm -rf ~/.config/gtk-3.0
```

## Notes

- **Copy first, then modify:** Always copy files to `config/` before updating Nix modules
- **Git commit per app:** One commit per migration for easy rollback
- **Backups:** Dotfiles repo is the backup — don't delete until verified
- **Testing:** After each change, run `sudo nixos-rebuild switch --flake .#<hostname>`
- **Rollback:** `sudo nixos-rebuild switch --rollback` if something breaks
- **Spaces in filenames:** Makima configs have spaces — test git handles them correctly
